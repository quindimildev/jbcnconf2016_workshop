Introduction

Are you ready? Maybe not! For attending this workshop is compulsory having a minimal environment requirements installed. It's very important cause we are not going to do the following during the workshop.

Environment requirements

Operating System

You can use Windows, Linux or OSX. So if you are a Windows lover, don't worry, but Linux/OSX is better :D

JVM

Grails works over a JVM, so at least JDK 1.7 have to be installed, feel free to use it, but we will use the 1.8 version instead.

Grails

We are going to use the lastest Grails version. The 3.1.8 version.

Manual installation

Install Grails is very easy, and even easier if you follow the Grails Documentation placed at: http://docs.grails.org/latest/guide/gettingStarted.html#requirements Don't forget to download the 3.1.8 Grails version.

Installing Grails with a manager tool (Linux/OSX)

If you are installing Grail over Linux or OSX you can use sdkman.io It's the best an easy way to do it. How?

  • Install SDKMan following instructions placed at: http://sdkman.io
  • After install SDKMan, run on the terminal: sdk install grails 3.1.8, and that's all!

Check if Grails has been installed

Just run grails -version on the terminal. You will see something similar to this:

    $ grails -version
    | Grails Version: 3.1.8
    | Groovy Version: 2.4.6
    | JVM Version: 1.8.0_77

Database

We are going to see how to connect Grails with a MySQL database. If you have a MySQL Server installed on your machine will be great, but it's not necessary. If you do:
